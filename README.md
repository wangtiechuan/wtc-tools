## Development

```bash
$ sudo n stable

$ nrm use tabobao

# install dependencies
$ pnpm i

# develop library by docs demo
$ pnpm start

# build library source code
$ pnpm run build

# build library source code in watch mode
$ pnpm run build:watch

# build docs
$ pnpm run docs:build

```
